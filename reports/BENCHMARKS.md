# Sudoku Solver/Generator -- Benchmark Results

Regenerated by `benchmarks/bench.py`. Python 3.10.12, single-threaded, CPython, no PyPy/JIT.

### Solve time on known hard puzzles

| Puzzle | Solve time |
|---|---|
| AI Escargot (Arto Inkala, 2006 -- famously not solvable by singles alone) | 38.99 ms |
| World's Hardest Sudoku (Arto Inkala, 2012) | 43.59 ms |
| Empty board (81 blanks -- worst case for search depth) | 7.85 ms |

### Generation time + achieved clue count by difficulty
